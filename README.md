## ğŸ§® How It Works

The Fibonacci sequence is defined by the recurrence:

```
F(n) = F(n-1) + F(n-2)
```

With user-defined values:
- `F(0) = a`
- `F(1) = b`

Each subsequent number is the sum of the previous two.
